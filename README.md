# Abalone Data Analysis Project
Data analysis and visualization project using the Abalone dataset with Python, Pandas, Matplotlib, and Seaborn in Google Colab.
This project focuses on analyzing the Abalone dataset using Python in Google Colab. The goal is to explore the dataset, perform data cleaning, create meaningful visualizations, and extract insights.

# Project Objectives
- Perform data cleaning and preprocessing  
- Handle categorical data for analysis  
- Explore dataset using statistics  
- Visualize patterns and relationships  
- Generate insights from real-world data

# Dataset Information
The dataset contains physical measurements of abalones such as:
- Sex (M, F, I)  
- Length  
- Diameter  
- Height  
- Whole Weight  
- Shucked Weight  
- Viscera Weight  
- Shell Weight  
- Rings (used to estimate age)  
- Age is calculated using:
    Age = Rings + 1.5

# Key Steps Performed
1. Data Loading and Exploration  
2. Data Cleaning and Column Renaming  
3. Feature Engineering (Age Calculation)  
4. Handling Categorical Data  
5. Data Visualization:
   - Age Distribution  
   - Count Plot (Sex)  
   - Correlation Heatmap  
   - Scatter Plot (Length vs Age)  

# Key Insights
- Age is strongly related to physical features like length and weight  
- Most abalones fall within a specific age range  
- Certain measurements show high correlation with each other
